## 🚀 Features

- 🌐 Fetches real-time weather data
- 🔍 Search by city name
- 🌡️ Displays temperature, weather condition, humidity, and wind speed
- 📱 Responsive design for desktop and mobile
- 📦 Built with Vanilla JavaScript and Fetch API

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- OpenWeatherMap API
